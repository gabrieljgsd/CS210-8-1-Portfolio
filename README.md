# Chada Tech Clocks

## Project Summary

This project involved developing a dual clock application that displays both a 12-hour and a 24-hour clock. The application allows users to add hours, minutes, and seconds while maintaining accurate time calculations and updating both clock formats simultaneously.

## What I Did Well

I successfully applied object-oriented programming principles to organize the project into multiple files and classes. The program accurately handles time calculations and provides a clear menu-driven interface for user interaction.

## Areas for Improvement

If I were to revise this project, I would improve the user interface and add additional input validation to prevent invalid user entries. I would also explore adding graphical elements to improve usability.

## Challenges and Solutions

One of the biggest challenges was ensuring that the 12-hour and 24-hour clocks remained synchronized after time updates. I overcame this by carefully testing different scenarios and validating the logic used for updating hours, minutes, and seconds.

## Transferable Skills

This project strengthened my skills in C++ programming, debugging, object-oriented design, and problem solving. These skills will be valuable in future software development projects and professional environments.

## Maintainability, Readability, and Adaptability

The project uses separate source and header files, meaningful variable names, and organized code structure to improve readability and maintainability. The design can be expanded in the future to support additional time formats or graphical interfaces with minimal modification.
